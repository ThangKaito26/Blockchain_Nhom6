Ứng dụng công nghệ Blockchain để giải quyết bài toán chứng chỉ giả mạo trong giáo dục

🔗

Dữ liệu bất biến

Một khi đã ghi lên Blockchain, không ai có thể sửa đổi hay xóa dữ liệu chứng chỉ, kể cả admin.

🌐

Minh bạch 100%

Mọi giao dịch cấp/thu hồi chứng chỉ đều công khai, có thể tra cứu trên Blockchain Explorer.

📄

Lưu trữ IPFS

File PDF gốc được lưu trên mạng IPFS phi tập trung, không phụ thuộc vào server trung tâm.

🔐

Xác thực bằng Hash

Dùng SHA-256 để tạo dấu vân tay số của file PDF. Chỉ cần upload file là biết ngay thật hay giả.

📱

QR Code nhanh

Mỗi chứng chỉ có một QR code riêng. Quét bằng điện thoại để xác thực chỉ trong vài giây.

🛡️

Quản lý trạng thái

Admin có thể thu hồi chứng chỉ bất kỳ lúc nào. Trạng thái "Revoked" hiển thị ngay lập tức.

Quy trình hoạt động

1️⃣

Admin Upload

Admin tải file PDF + nhập thông tin người nhận lên hệ thống

2️⃣

Tạo Hash

JavaScript tính SHA-256 hash của file PDF – dấu vân tay số

3️⃣

Lưu Blockchain

Hash + thông tin được ghi vĩnh viễn lên Smart Contract

4️⃣

Xác thực

Người dùng upload file → so sánh hash → kết quả tức thì

Bắt đầu xác thực ngay

Nhập mã chứng chỉ hoặc upload file PDF để kiểm tra tính xác thực
