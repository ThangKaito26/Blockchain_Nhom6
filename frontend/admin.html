Yêu cầu quyền Admin

Chức năng này chỉ dành cho tài khoản Admin (địa chỉ ví đã deploy contract). Vui lòng kết nối MetaMask với đúng tài khoản admin.

📖 Hướng dẫn cho đồ án:

  1. Chạy: npx hardhat node
  2. Deploy: npx hardhat run scripts/deploy.js --network localhost
  3. Thêm mạng vào MetaMask: RPC http://127.0.0.1:8545, Chain ID 31337
  4. Import private key tài khoản #0 từ output của hardhat node
